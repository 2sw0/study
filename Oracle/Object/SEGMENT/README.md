SEGMENT別情報
---
```sql
SELECT SEGMENT_TYPE ,
       SEGMENT_NAME ,
       TABLESPACE_NAME ,
       TRUNC((SUM(BYTES)/1024)/1024,2) as "MB"
FROM DBA_SEGMENTS   
WHERE SEGMENT_TYPE IN ('TABLE', 'INDEX', 'TABLE PARTITION', 'TABLE SUBPARTITION')
  --AND OWNER = 'owner'
GROUP BY SEGMENT_TYPE , SEGMENT_NAME, TABLESPACE_NAME
ORDER BY 1 desc, "MB" desc;
```
